# Contact Page with Azure Function

## Overview
Static contact form sending email using Azure Communication Service.

## Features
- Name, Email, Subject, Message fields
- Email validation
- HTML tag blocking for security
- Environment variables for configuration

## Setup

### Azure Function Environment Variables

ACS_CONNECTION_STRING  
RECIPIENT_EMAIL  
SENDER_EMAIL  

### Deployment
1. Deploy Azure Function
2. Deploy static HTML
3. Replace function URL in index.html
